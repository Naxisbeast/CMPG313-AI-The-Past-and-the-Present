# CMPG313 - Lab 3: AI The Past and the Present

## Overview
This project compares rule-based AI (ELIZA chatbot) with a modern 
Large Language Model (LLM) using the Hugging Face Inference API.
Both systems are tested using the same prompts to observe differences 
in pattern matching vs contextual understanding.

## Project Structure
- eliza.py — Rule-based ELIZA chatbot implementation
- LLM.py — Modern LLM using Hugging Face Inference API
- chat_comparison.py — Side-by-side comparison of both systems
- screenshots/ — CMD execution and conversation screenshots
